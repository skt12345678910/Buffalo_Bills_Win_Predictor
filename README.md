This project is a live win probability predictor for the Buffalo Bills. It uses computer algorithms to study how football games unfold play by play. Many basic models cheat by testing themselves on parts of games they have already seen. To avoid this, this project splits data cleanly by entire games. This forces the AI to guess on completely fresh, unseen matchups, mimicking how professional platforms like ESPN make predictions. The system trained on a dataset of 100,000+ real NFL offensive plays from the 2024 and 2025 seasons. The model studied all 32 NFL teams to learn how football works, and then tested its skills exclusively on the Buffalo Bills to achieve a true accuracy rate of 77.94%.Instead of just looking at the scoreboard, the AI blends multiple in-game factors together to make its decisions. It evaluates clock pressure by calculating how a point lead becomes exponentially safer as time runs out. It also tracks fading expectations, which measures how much pre-game Vegas odds matter early in the game versus late in the game. Finally, it monitors team momentum by tracking live offensive efficiency and drive history across the match.Computer models can sometimes get confused during rare, extreme situations, such as a team having only 3 seconds left to score a touchdown from 90 yards away. To fix this, the interactive dashboard includes a custom safety gatekeeper code. If the app detects an impossible physics scenario, it automatically overrides the confused AI and sets the win chance to a logical 100% or 0%. This hybrid approach successfully blends machine learning with real-world logic. Anyone can test the application by clicking the orange Open in Colab badge at the top of the page. Once inside, select Runtime from the top menu and click Run All to start the engine. Users can then scroll to the very bottom cell to use the live form sliders and test any game scenario.
