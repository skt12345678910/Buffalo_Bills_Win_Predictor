This project is a live win probability predictor for the Buffalo Bills. It uses an XGBoost Classifier and a 5-Fold Grid Search to study how football games unfold, play by play. The system was trained on a dataset of 64,000 NFL offensive plays from the 2024 and 2025 seasons. The model learned from all 32 NFL teams to broaden its learning set, and then was tested exclusively on the Buffalo Bills' offensive plays. This model peaked at an accuracy rate of 78.35%, making it comparable to the Pro Football Reference (PFR) model.

Instead of just looking at the scoreboard, the model blends 14 multiple in-game factors together to make its decisions. Some of those features are advanced, factoring in clock pressure, offense efficiency, and fading odds, as examples. The model has a limitation during the final plays of games, where it doesn't value down and distance enough, while also overvaluing factors like point spread and game location. This is protected through a custom safety code that ensures late-game scenarios where the winner is practically settled give an accurate prediction of 99.9% or 0.01% win probability.

Anyone can test the application by opening the Buffalo_Bills_Win_Predictor.ipynb file and clicking the Open in Colab link at the top of the page. Once inside, you can select Runtime from the top menu and click Run All to start the engine. You can then scroll to the very bottom cell to use the live form input and test any game scenario.
