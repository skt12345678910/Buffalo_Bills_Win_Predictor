This project is a live win probability predictor for the Buffalo Bills. It uses computer algorithms to study how football games unfold play by play. The system was trained on a dataset of 64,000 NFL offensive plays from the 2024 and 2025 seasons. The model learned from all 32 NFL teams to broaden its learning set, and then was tested exclusively on the Buffalo Bills' offensive plays. This model peaked at an accuracy rate of 78.35%. 
Instead of just looking at the scoreboard, the model blends 14 multiple in-game factors together to make its decisions. Some of those features are advanced, factoring in clock pressure, offense efficiency, and fading odds as some examples. The model has a limitation during the final plays of games, where it doesn't value down and distance enough, while also overvaluing factors like point spread and game location. This is protected through a custom safety code that ensures late-game scenarios where the winner is practically settled give an accurate prediction of 99.9% or 0.01% win probability. 
Anyone can test the application by clicking the orange Open in Colab badge at the top of the page. Once inside, select Runtime from the top menu and click Run All to start the engine. Users can then scroll to the very bottom cell to use the live form sliders and test any game scenario.
